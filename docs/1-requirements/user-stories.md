# User Stories - InduLex

## Backlog de Historias de Usuario

| ID | Historia de Usuario | Criterio de Aceptación |
|----|----------------------|------------------------|
| US01 | Como **usuario**, quiero **iniciar sesión con Google** para no tener que recordar otra contraseña. | - Botón de Google funcional.<br>- Persistencia en localStorage.<br>- Redirección a Home tras éxito. |
| US02 | Como **comprador**, quiero **filtrar productos** para encontrar rápidamente lo que busco. | - Filtros por categoría (Hoodies, Tees, etc.).<br>- Actualización dinámica del catálogo. |
| US03 | Como **usuario**, quiero **editar mi perfil** para mantener mi dirección de envío actualizada. | - Formulario de edición funcional.<br>- Validación de campos en cliente y servidor. |
| US04 | Como **comprador**, quiero **ver mi carrito** antes de pagar para confirmar mi pedido. | - Modal de carrito con resumen de precios.<br>- Posibilidad de cambiar cantidades. |
| US05 | Como **usuario**, quiero **cambiar mi avatar** para personalizar mi experiencia. | - Generador de avatares (DiceBear) integrado.<br>- Cambio instantáneo en Navbar. |
| US06 | Como **administrador**, quiero **gestionar el stock** para evitar vender productos agotados. | - Endpoint de actualización de producto funcional.<br>- Validación de stock en el checkout. |
